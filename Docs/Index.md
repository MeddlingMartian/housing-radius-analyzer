# Housing Radius Analyzer

> **Analyze housing listings within a custom radius — fast, visual, and open source.**

Copyright (C) 2026 Tyler Blankenship · Rae · Lucas · Daloze Benoit (Eregon)

---

## What It Does

The **Housing Radius Analyzer** lets you drop a pin anywhere on a map and instantly surface all housing options within a configurable mile/km radius. It aggregates listing data, scores properties by commute time, price, and walkability, and exports results to CSV or JSON.

## Features

- 📍 **Radius Search** — define any center point and distance
- 🏠 **Multi-source Listings** — pulls from public datasets
- 🚗 **Commute Scoring** — weights results by drive/transit time
- 📊 **Export** — CSV, JSON, or shareable link
- 🗺️ **Interactive Map** — Leaflet.js powered visualization

## Quick Start

```bash
git clone https://github.com/MeddlingMartian/housing-radius-analyzer.git
cd housing-radius-analyzer
npm install
npm start
```

Then open `http://localhost:3000`.
